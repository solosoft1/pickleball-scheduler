Testing
=======

This project includes Jest unit tests for `rr_scheduler.js`.

Quick start
-----------

1. Install dev dependencies:

```bash
npm install
```

2. Run tests:

```bash
npm test
```

Notes
-----
- Tests use Jest and assume CommonJS `require` exports from `rr_scheduler.js`.
- If you prefer a different test runner (Mocha/AVA), update `package.json` and test files accordingly.
